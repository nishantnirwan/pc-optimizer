The Core Fix

Every command runs invisible.

The #1 complaint with optimizer tools is the constant flash of black terminal windows. Astranyx kills that at the OS level using two Windows APIs on every single subprocess call.

CREATE_NO_WINDOW (0x08000000)
Process creation flag that tells Windows not to allocate a console window for the child process.
STARTUPINFO + SW_HIDE
Forces the startup window state to hidden even if the process tries to show one.
subprocess_helper.py 
# Suppress ALL child process windows
_NO_WINDOW = 0x08000000

def _si():
    si = subprocess.STARTUPINFO()
    si.dwFlags |= subprocess.STARTF_USESHOWWINDOW
    si.wShowWindow = subprocess.SW_HIDE
    return si

def run_ps(script):
    subprocess.run([
        "powershell", "-NoProfile",
        "-WindowStyle", "Hidden",
        "-Command", script
    ],
    creationflags=_NO_WINDOW,
    startupinfo=_si())
